## üé® Easy Edits to Make

Open `index.html` in any text editor (VS Code, Notepad++, etc.)

### Change Colors:
Find this in the `<style>` section:
```css
--color-primary: #1E3A5F;  /* Navy - change this */
--color-accent: #D4AF37;   /* Gold - change this */
```

### Change Church Name:
Find and replace: "True Gospel Evangelistic Ministry"

### Change Photos:
Replace Unsplash URLs with your own:
```html
<img src="YOUR-PHOTO-URL.jpg" alt="...">
```

### Add Go High Level Webhook:
Find this JavaScript section and add your webhook URL:
```javascript
// TODO: Add your GHL webhook URL here
fetch('YOUR_WEBHOOK_URL', {...})
```
