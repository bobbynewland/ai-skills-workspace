## Commands
```bash
# Transcribe file
python3 /root/.openclaw/workspace/skills/whisper/whisper_skill.py transcribe video.mp4

# YouTube download + transcribe
python3 /root/.openclaw/workspace/skills/whisper/whisper_skill.py url <youtube_url>

# Check status
python3 /root/.openclaw/workspace/skills/whisper/whisper_skill.py status
```
